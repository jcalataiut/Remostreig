Monte Carlo methods in the most basic form is used to approximate to a result aggregating repeated probabilistic experiments. For instance, to find the true probability of heads in a coin toss repeat the coin toss enough (e.g.100 times) and calculate the probability by dividing number of heads to the total number of experiments.

Do a function to simulate this experiment and return this simulation with (C, +). The number of tosses and the probabilities must be a parameters. The function must return a list with:

1.The simulated sequence. 2.The empirical probabilities for tails and crosses. 3.A ggplot2 plot with the empirical results for the accumuled empirical values

#Valors 1=C i 0=X

simulacio <- function(n,p){

  sequen <- sample(c(0,1), size=n, replace = TRUE, prob = c(1-p,p))
  prob_C <- sum(sequen==1)/n
  prob_X <- sum(sequen==0)/n
  t<-seq_len(n)
  acum <- cumsum(sequen)/t
  
  df <- data.frame(x = t, y = acum)

  plotseq <- ggplot(df, aes(x = x, y = y)) +
    geom_line()+
    ylim(0,1)+
    geom_hline(yintercept = p, linetype="dashed", color="red")
  
  return(list(factor(sequen,levels=c(0,1), labels=c("+","C")), prob_C, prob_X, plotseq))
}

sim_cumple_equitativa <- function(rep,n){
  t<-1:365
  mostra <- apply(t(replicate(rep,sample(t, size=n, replace = TRUE, prob = rep(1/365,365)))),MARGIN=1,FUN=function(x) {length(unique(x))!=n})
  prob<- sum(mostra)/rep
  return(prob)
}

sim_cumple_lineal <- function(rep,n){
  t<-1:365
  vector_prob<-t/sum(t)
  mostra <- apply(t(replicate(rep,sample(t, size=n, replace = TRUE, prob = vector_prob))),MARGIN=1,FUN=function(x) {length(unique(x))!=n})
  
  prob<- sum(mostra)/rep
  return(prob)
}

montecarlo_integral <- function(limits_integral, f, rep, epsilon=0.1){
  rep<-10000
  #trobar mxim amb simulacio
  limits_integral<-c(0,1)
  sequen<-seq(limits_integral[1],limits_integral[2], by=0.001)
  epsilon<-0.1
  maxim <- max(f(sequen))+epsilon
  
  x<-runif(rep,min=limits_integral[1],max=limits_integral[2])
  y<-runif(rep,min=0, max=maxim)
  factor_cor <- (limits_integral[2]-limits_integral[1])*maxim
  area <- length(y[(y<=f(x))])/rep*factor_cor
  
  dt<- data.frame(x,y)
  p<-ggplot(dt, aes(x=x,y=y))+
    geom_point(aes(colour= y<=f(x)), show.legend = FALSE)+
    xlim(c(limits_integral[1],limits_integral[2]))+
    ylim(c(0,maxim))+
    geom_function(fun=f,colour="black",linetype=1)
  
  return(list(area,p))
}

  1. Proveu analticament que \(E(X_n)=n(p-q)\) i que \(V(X_n)=4npq. Calculeu quina s la distribuci assipttica del procrs segons el Teorema Central del Lmit.\)

Si n=0, \(E(X_n)=E(0)=0\) i la identitat es verifica trivialment. Ara suposem que \(n>0\),

\[ E(X_n)=E( \sum_{i=1}^n Y_i )= \sum_{i=1}^n E(Y_i) = n \cdot (p-q) \] la segon igualtat es dedueix del fet que les variables dicotmiques \(Y_i\) estan iid. Endems, sha tingut en compte que \(E(Y_i)=1 \cdot p + (-1) \cdot q = p-q\)

Aleshores, \(\forall n\) amb \(n>0\) es verifica que \(E(X_n)=n\cdot(p-q)\). Per que respecta a la varincia, sutilitzar el fet que \(V(Y_n)=E(Y_n^2)-E(Y_n)^2\) i que \(E(Y_n^2)=1^2 \cdot p + (-1)^2 \cdot q = p+q\). Per tant, \(V(Y_n)=(p+q)-(p-q)^2\) en efecte, de la mateix forma que en el clcul anterior, per \(n=0\) els resultat es verifica trivialment i suposant \(n>0\) es t:

\[ V(X_n)=V( \sum_{i=1}^n Y_i )= \sum_{i=1}^n V(Y_i) = \sum_{i=1}^n [ E(Y_n^2)-E(Y_n)^2]= \sum_{i=1}^n (p+q)-(p-q)^2 = \\ = n(p+q)-n(p-q)^2 = n(p+1-p) - n (p-(1-p))^2=n-n(2p-1)^2 = n - n(4p^2+1-4p) = \\ = n - 4np^2 - n + 4np = 4np(1-p) = 4npq \] Recordarem el resultat del Teorema Central del Lmit:

Si \(X_1, X_2,..., X_n\) son variables aleatories iid amb esperana \(\mu=E(X_i)\) i varincia \(\sigma^2=V(X_i)\) \(\forall i=1,2,...,n\) es verifica que la variable suma \(Y=X_1+X_2+...+X_n\) (quan \(n \rightarrow \infty\)) es pot aproximar per una distribuci normal amb esperana suma de esperances i varincia com suma dd varincies. s a dir;

\[ Y=X_1+X_2+...+X_n \sim N\left(\sum_{i=1}^n \mu_i, \sqrt{\sum_{i=1}^n \sigma_i^2}\right) \]

En conclusi, el TCL afirma que si la v.a. X s la suma duna gran quantitat de variables aleatria iid, aleshores X es distribueix aproximadament normal.

El mtodo de Montecarlo sutilitza para estimar probabilitats i valors esperats mitjanant la generaci de nmeros aleatoris que sigueixen una distribucin especfica. La idea principal es realitzar mltiples simulacions i promigar els resultados per a obtindre una estimaci ms precisa.

En el notre cas, consistir en simular resultats de sucessions de variables aleatories dicotmiques iid amb probabilitat dxit \(p\). probabilitat <- function(n,p,rep,tipo, valor){
  if(tipo=="menor"){
    fre <- sum(apply(matrix(replicate(rep,sim_rw(n,p)[n]), ncol=1),MARGIN=1,FUN=function(x) {x<valor}))
    return(fre/rep)
  }
  
  if(tipo=="major"){
    fre <- sum(apply(matrix(replicate(rep,sim_rw(n,p)[n]), ncol=1),MARGIN=1,FUN=function(x) {x>valor}))
    return(fre/rep)
  }
}
